
## Description

GHA Backend service repository.

## Installation

```bash
$ yarn install
```
## .env file 
Create .env file with reference to env-example

## Generate Prisma schema

```bash
$ npx prisma generate
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```


## Accessing swagger

visit http://localhost:3000/docs/#/
